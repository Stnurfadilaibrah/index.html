Pengalaman Selama Kuliah

Saya telah belajar banyak hal selama mengikuti perkuliahan di PTIK. Selain belajar teori, saya juga mendapatkan pengalaman praktis yang sangat berharga melalui berbagai proyek, diskusi, dan kegiatan lapangan. Berikut beberapa mata kuliah dan pengalaman yang sangat berkesan selama kuliah:

Pemrograman Web

Mata kuliah ini mengajarkan tentang pengembangan web menggunakan HTML, CSS, JavaScript, dan framework seperti Bootstrap. Saya belajar membangun halaman web yang responsif, interaktif, dan berfokus pada pengalaman pengguna.

Basis Data

Pada mata kuliah ini, saya belajar tentang pengelolaan data secara efisien menggunakan sistem basis data seperti MySQL dan PostgreSQL. Saya juga mengerjakan proyek untuk mendesain dan mengimplementasikan skema basis data yang kompleks.

Jaringan Komputer

Di mata kuliah ini, saya belajar tentang konsep jaringan komputer, mulai dari topologi jaringan, protokol, hingga pengelolaan jaringan menggunakan perangkat keras dan lunak. Pengalaman ini sangat membantu dalam memahami bagaimana internet bekerja.

Pengembangan Aplikasi Mobile

Saya belajar mengembangkan aplikasi mobile menggunakan framework seperti Flutter dan React Native. Kami diajarkan cara membangun aplikasi lintas platform yang dapat berjalan di Android dan iOS.

Kecerdasan Buatan (AI)

Mata kuliah ini memperkenalkan konsep dasar kecerdasan buatan, termasuk algoritma pembelajaran mesin (machine learning) dan pemrosesan bahasa alami (natural language processing). Saya juga berkesempatan membuat proyek sederhana berbasis AI.

Manajemen Proyek TI

Pada mata kuliah ini, saya belajar bagaimana merencanakan, mengelola, dan mengimplementasikan proyek teknologi informasi dari awal hingga akhir. Kami juga belajar tentang metodologi seperti Agile dan Scrum untuk pengelolaan proyek yang lebih efisien.
